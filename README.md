## @nivalis/app-template

### Monorepo layout

```text
apps/
  web/    # next.js app template
packages/ # shared packages (coming next)
```

### Install

```bash
pnpm create next-app --example https://github.com/nivalis-studio/app-template
```

### Development

```bash
pnpm dev
```

### Create a new neon database

```bash
bunx neondb --yes
```
